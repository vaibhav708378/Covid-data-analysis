# 🦠 COVID-19 Data Analysis Project

## 📌 Overview
This project focuses on analyzing COVID-19 data in India using Python. It explores trends in confirmed cases, recoveries, deaths, and vaccination progress across different states.

The goal is to extract meaningful insights from real-world datasets using data analysis and visualization techniques.

---

## 📂 Dataset
The project uses the following datasets:
- `covid_19_india.csv` → Contains daily COVID-19 case data
- `covid_vaccine_statewise.csv` → Contains vaccination data by state

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Features
- Data cleaning and preprocessing
- Analysis of confirmed, recovered, and death cases
- State-wise comparison of COVID-19 impact
- Vaccination trend analysis
- Visualizations using graphs and charts

---

## 📈 Visualizations Included
- Line graphs for case trends
- Bar charts for state comparison
- Vaccination progress charts
